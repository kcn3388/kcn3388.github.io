快速开始

创建一篇新文章

$ hexo new "My New Post"

更多资讯：写作

运行Hexo服务器

$ hexo server

-i 参数指定IP，-p参数指定端口

例子：在127.0.0.1:8080上启动服务器：

$ hexo server -i 127.0.0.1 -p 8080
